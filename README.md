# 我去不早说

Claude Code 会话监控器，当别人用 Claude 提问时，3秒后弹窗提醒。

## 使用

双击 `start.bat`，右下角出现橙色 "C" 图标。

**右键菜单：**
- 暂停 / 恢复
- 退出

## 开机自启

双击 `add_startup.bat`

## 文件

| 文件 | 说明 |
|------|------|
| `woqu.py` | 主程序 |
| `icon.ico` | 图标 |
| `start.bat` | 启动 |
| `add_startup.bat` | 添加开机自启 |
| `remove_startup.bat` | 移除开机自启 |
